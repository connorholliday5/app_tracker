# grad-app-tracker

Grad App Tracker is a lightweight application tracking system built with Python and Streamlit to manage graduate school and research job applications.

The goal of this project is to create a structured, automated dashboard that tracks application status, interview progress, and follow-up reminders.

The system automatically flags applications that require follow-up when 14 days have passed since the application date.

---

## Core Features

- Track graduate school and research job applications
- Application status tracking
- Interview stage tracking
- Automated follow-up detection
- Dashboard metrics
- CSV / Excel export
- Streamlit dashboard UI
